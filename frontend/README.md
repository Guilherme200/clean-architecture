# Application Task Minimal Starter

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Tests

Build the application for tests:

```bash
# npm
npm run test
```

## Production

Build the application for production:

```bash
# npm
npm run build
```
